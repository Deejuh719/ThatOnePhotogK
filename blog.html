What Do You Do As a Web Developer?

So, what do I do? I am an aspiring web developer. What does that mean? It means I make web pages look all pretty and functional. I
create things like this site that you're looking at. I use something called HTML (that stuff we learned to make our Myspaces pretty, but then instantly forgot)

Turns This...

Right now, this HTML full of funky symbols and words don't seem like much of anything other than another language but...

...Into This

The mumbo jumbo of letters turns into something like this. Or the entire web page that you're scrolling around on!

All of this code is what makes up everything you see from the moment you open an app on your phone, to when you Google how to make that recipe you lied and said you totally know how to make. In three weeks of a four month course, I have already learned so much fantastic stuff that has jogged my memory back in to gear, and it's so exciting! Just as an example of some things that I learned to do that I didn't quite remember before was how to make tables using <table> and inserting various pieces of information. Some of which helped make this very layout look clean (until I learn how to better clean it up later). I also learned how to make a "Style Sheet" and call on it when I need it so that way there's less clutter in the actual body of my code. All of this information alone was useful enough to help me make this awesome looking site that I'm 99% sure I overacheived on. But who would I be if I didn't over-accomplish something?

Some New Things I've Learned

More things I have learned in class now consist of animations and Bootstrap™. I had learned animations in middle school when I had first began learning web development, but that was done in a whole separate app at the time. I didn't have the luxury of Bootstrap™ in the early 2000's. I had to go into a seprate program (Adobe Flash™, just to show my age) and create the effect that way, then import it to my page. With Bootstrap™, I can now animate in my page easily without having to take fifty-eleven steps to get there.
